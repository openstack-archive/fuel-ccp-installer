Fuel CCP installer
==================

Fuel CCP installer is a wrapper for a
`Kargo <https://github.com/kubernetes-incubator/kargo>`_ installer.
It uses Ansible CM tool to install Kubernetes clusters.
It also provides some extra tools and playbooks.
